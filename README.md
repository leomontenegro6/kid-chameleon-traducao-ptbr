# [SMD] Kid Chameleon - Tradução PT-BR

Este é um projeto de tradução do jogo Kid Chameleon, de Mega Drive, para português do Brasil. O projeto foi feito com base na versão americana do jogo, usando por base o repositório do [Kid Chameleon Disassembly](https://github.com/sonicretro/kid-chameleon-disasm).

Kid Chameleon é um jogo de plataforma, desenvolvido pela Sega em 1992. É um dos primeiros da vida útil do Mega Drive, contendo uma quantidade enorme de telas e fases pros jogos da época.
